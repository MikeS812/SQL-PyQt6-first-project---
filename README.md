# SQL-PyQt6-first-project---
My first project on yandex.
Как запустить код:
Откройте PyCharm или другие IDE
убедитесь что в директории рядом с 3 файлами классов лежат YandexPres.ui, YandexTool.ui, Teen_db.sqlite, recomend.txt
Установите зависимости из файла requirements.txt
Запустите программу 

Сборка осуществлялась благодаря pyinstaller
команда: pyinstaller --onefile --noconsole YaPres.py

Как работает приложение:

1. Для добавления студента заполните все поля и нажмите 'Добавить'.
   Ф.И.О нужно вводить в заданном порядке с большой буквы через пробел.
   Номер обязан быть формата "(любой код страны)-XXX-XXX-XX-XX". Это проверяется!
   Курс также должен быть числом в диапазоне 1-4.
2. Для удаления студента выберите критерий, введите значение и причину.
    Критерий обязателен
3. Просмотр всех студентов доступен во вкладке 'Журнал'.
4. В настройках можно выбрать стиль приложение а так-же в настройках есть ссылка на восстановление приложения если какие-то из файлов утеряны.
